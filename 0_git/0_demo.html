1. git 连接 github

    登录 github 后，在指定项目的按钮 Clone or download ，获取项目连接，在 git 界面输入以下代码：

    git clone git@github.com:michaelliao/bootstrap.git

  2. git 连接 github

    ssh-keygen -t rsa -C "your_email@youremail.com"

    PTY allocation request failed on channel 0
